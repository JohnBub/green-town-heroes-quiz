Exercices d'Entraînement

Exercice 1 : Quel est le plus petit nombre sur un octet ?
Réponse : Le plus petit nombre est 0. En binaire, cela s'écrit 0000 0000. Tous les transistors sont "éteints".
Exercice 2 : Quel est le plus grand nombre sur un octet ?
Réponse : Le plus grand nombre est 255. En binaire, cela s'écrit 1111 1111.
Calcul : 128 + 64 + 32 + 16 + 8 + 4 + 2 + 1 = 255.
Exercice 3 : Convertir 1010 1010 en décimal
Réponse : Calcul : 128 + 32 + 8 + 2 = 170.

Référence Binaire

Table des Puissances de 2
Positions 8-bits : 128, 64, 32, 16, 8, 4, 2, 1

Méthode : Utilisez toujours la table des puissances de 2 pour les conversions.
Placez un 1 sous chaque puissance qui s'additionne pour obtenir votre nombre cible.
Conversion Binaire vers Décimal
Étape 1 : Écrivez les puissances de 2 : 128, 64, 32, 16, 8, 4, 2, 1
Étape 2 : Pour chaque 1 dans le nombre binaire, ajoutez la puissance de 2 correspondante
Étape 3 : Sommez toutes les valeurs où il y a un 1

Exemple : 1100 1001 = 128 + 64 + 8 + 1 = 201
