Le groupe d'octets

Un seul bit est tres limite. C'est pourquoi les ordinateurs regroupent les bits par paquets de 8. Ce groupe s'appelle un octet.

  • Definition : 1 octet = 8 bits
  • Notation : On utilise B (exemple : 1 B = 8 bits)
  • Capacite : Avec 8 bits, on peut representer 256 valeurs differentes (0 a 255)
Multiples de l'octet

Comme les fichiers sont souvent tres grands, on utilise des prefixes :

Unite Symbole Valeur approximative
Octet B 1 octet
Kilooctet KB 1 000 octets (petit texte)
Megaoctet MB 1 million d'octets (photo)
Gigaoctet GB 1 milliard d'octets (film HD)
Teraoctet TB 1 000 milliards d'octets (disque dur)
